# Escape from Bear

## ゲームの内容
街にはびこる熊と戦うか逃げるかして、道の先にある車までたどり着こう

![画面イメージ](docs/images/game_image01.png)

## 操作説明

- 矢印キー、若しくはWASDキーで前後左右にキャラクターを移動する
- 左クリックで攻撃をする
